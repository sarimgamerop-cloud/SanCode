## About San

San is a dynamically and strongly typed interpreted programming language, that enforces strong immutability and prevents accidental reassignments in the long run.

San is an extremely simple programming language, with python as its backend. It has the potential to become the best programming language for beginners who are just starting to learn programming. Its dynamic and strong typing allows students to learn about what they should do, and what they should not. 

It also uses conventions from multiple languages, so if you learn San, you learn the following programming language conventions simultaneously:

- Python 
- Java
- C
- C++
- Rust 
- Javascript
